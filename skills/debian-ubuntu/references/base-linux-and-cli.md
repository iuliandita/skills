# Base Linux and CLI

Use this for basic inspection commands on Debian-style systems.

## Common commands
- `journalctl`
- `dmesg`
- `lsblk`
- `findmnt`
- `ps aux`
- `ss -tulpn`
- `update-alternatives --display name`

## Notes
- Keep diagnostics boring and factual.
- Do not hide useful errors with blanket redirection.
